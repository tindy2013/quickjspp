Detailed Description

JSValue with RAAI semantics.

A wrapper over (JSValue v, JSContext * ctx). Calls JS_FreeValue(ctx, v) on destruction. Can be copied and moved. A JSValue can be released by either JSValue x = std::move(value); or JSValue x = value.release(), then the Value becomes invalid and FreeValue won't be called Can be converted to C++ type, for example: auto string = value.as<std::string>(); qjs::exception would be thrown on error Properties can be accessed (read/write): value["property1"] = 1; value[2] = "2";
